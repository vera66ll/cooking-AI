## 🚀 使用流程

### 首次启动

1. **开通云开发**
   ```
   微信开发者工具 -> 云开发 -> 开通服务 -> 选择基础版
   ```

2. **获取环境ID**
   ```
   云开发控制台 -> 环境设置 -> 复制环境ID
   ```

3. **修改配置**
   ```typescript
   // miniprogram/app.ts 第10行
   env: 'your-env-id'  // 替换为实际环境ID
   ```

4. **上传云函数**
   ```
   右键 cloudfunctions 文件夹
   -> 选择当前环境
   -> 依次上传每个云函数
   -> 选择"上传并部署：云端安装依赖"
   ```

5. **初始化数据库**
   ```javascript
   // 在控制台执行
   wx.cloud.callFunction({ name: 'initDatabase' })
   ```

6. **运行测试**
   ```
   编译运行小程序
   -> 查看控制台日志
   -> 确认登录成功
   ```
