# 登录和登出功能说明

## 功能概述

本项目已实现完整的用户登录和登出功能，包括：
- 账号密码登录
- 微信一键登录
- 记住我功能
- 表单验证
- 登出确认对话框
- 响应式设计

## 功能特性

### 1. 登录页面特性

#### UI设计
- 简洁现代的界面设计
- 与项目整体UI风格一致（橙色主色调 #FF6B35）
- 渐变背景和圆角卡片设计
- 响应式布局，适配各种屏幕尺寸

#### 表单元素
- **手机号输入框**：支持11位手机号输入，带格式验证
- **密码输入框**：密码类型输入，最少6位
- **记住我选项**：勾选后保存登录信息
- **登录按钮**：带加载状态，防止重复提交
- **微信一键登录**：推荐使用，更便捷安全
- **忘记密码链接**：提示联系客服
- **注册链接**：预留注册功能入口

#### 表单验证
- 手机号非空检查
- 手机号格式验证（正则：`^1[3-9]\d{9}$`）
- 密码非空检查
- 密码长度验证（最少6位）
- 实时错误提示

#### 交互体验
- 输入时清除错误提示
- 登录过程显示"登录中..."状态
- 登录成功显示提示并跳转到首页
- 登录失败显示具体错误信息
- 已登录用户自动跳转到首页

### 2. 登出功能特性

#### 登出确认对话框
- 自定义模态框组件
- 优雅的动画效果
- 点击遮罩层可关闭
- 清晰的确认和取消按钮

#### 登出逻辑
- 清除本地存储的用户信息
- 清除记住的登录信息
- 重置全局用户状态
- 跳转到登录页面
- 友好的加载和成功提示

## 使用方法

### 访问登录页面

登录页面已设置为第一个页面，应用启动时会自动显示：
```json
// app.json
{
  "pages": [
    "pages/login/login",  // 第一个页面
    ...
  ]
}
```

如果用户已登录，登录页面会自动跳转到首页。

### 账号密码登录

1. 输入手机号（11位）
2. 输入密码（至少6位）
3. 可选：勾选"记住我"
4. 点击"登录"按钮

### 微信一键登录（推荐）

1. 点击"微信一键登录"按钮
2. 授权用户信息
3. 自动完成登录

### 登出操作

1. 进入"我的"页面
2. 点击底部的"退出登录"按钮
3. 在确认对话框中点击"确认"
4. 完成登出，跳转到登录页面

## 数据存储

### 本地存储键值

| 键名 | 说明 | 何时存储 |
|------|------|----------|
| `userInfo` | 用户信息对象 | 登录成功后 |
| `rememberedPhone` | 记住的手机号 | 勾选"记住我"时 |
| `rememberedPassword` | 记住的密码 | 勾选"记住我"时 |

### 全局数据

```typescript
// app.ts
App({
  globalData: {
    userInfo: null  // 当前登录用户信息
  }
})
```

## API对接

### 现有云函数

- `userLogin`: 用户登录（支持微信登录和账号密码登录）
- `getUserInfo`: 获取用户信息
- `updateUserInfo`: 更新用户信息

### 登录接口调用示例

```typescript
// 微信登录
const result = await CloudBase.userLogin(userInfo);

// 账号密码登录（需要扩展云函数）
const result = await CloudBase.callFunction('userLogin', {
  loginType: 'password',
  phone: '13800138000',
  password: '123456'
});
```

## 注意事项

### 微信小程序限制

1. **账号密码登录限制**：微信小程序对账号密码登录有审核限制，建议优先使用微信一键登录
2. **getUserInfo接口**：需要使用 `<button open-type="getUserInfo">` 才能获取用户信息
3. **隐私保护**：不要在日志中输出敏感信息（如密码）

### 安全建议

1. 生产环境应使用HTTPS
2. 密码应加密传输和存储
3. 建议使用token机制进行身份验证
4. 定期清理过期的登录信息

### 扩展建议

## 故障排查

### 登录页面不显示

检查 `app.json` 中是否正确注册了登录页面：
```json
{
  "pages": [
    "pages/login/login",
    ...
  ]
}
```

### 微信登录失败

1. 检查是否在真机上测试（微信登录需要真机）
2. 检查云开发环境是否正确初始化
3. 检查 `userLogin` 云函数是否已上传

### 登出后仍显示登录状态

1. 检查是否正确清除了 `userInfo`
2. 检查全局数据是否重置
3. 刷新页面或重启小程序
