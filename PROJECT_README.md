## 📝 使用说明

### 发布菜品
1. 点击底部"发布"按钮
2. 上传菜品图片（支持1-9张）
3. 填写菜品名称、选择菜系
4. 输入做法描述
5. 添加标签（可选）
6. 点击发布

### 收藏菜品
- 在首页点击菜品卡片右上角的心形图标
- 在"收藏"页面查看和管理收藏

### 加入菜单
1. 进入菜品详情页
2. 点击"加入菜单"按钮
3. 选择早餐/午餐/晚餐
4. 在"菜单"页面查看今日安排
